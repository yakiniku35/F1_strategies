賽道向量儲存與繪製總結

   你的賽道向量系統有三層架構：

   資料來源（優先順序）

     - FastF1 歷史資料 - 從真實比賽遙測提取 5000-10000 個 X,Y 座標點
     - 預設簡化佈局 - Monaco, Monza, Silverstone 等手動定義的點
     - 橢圓軌道生成 - 數學公式自動生成備用軌道

   繪製演算法（向量微積分）

     # 1. 從中心線計算切線（微分）
     dx = gradient(X)
     dy = gradient(Y)
     
     # 2. 取垂直向量
     nx = -dy  # 法向量
     ny = dx
     
     # 3. 計算賽道邊緣
     外緣 = 中心線 + 法向量 × (賽道寬度/2)
     內緣 = 中心線 - 法向量 × (賽道寬度/2)

   ⚡ 渲染優化

     - 批次渲染: Arcade ShapeElementList 一次性繪製所有線條
     - 智能快取: 只在賽道狀態改變（黃旗/紅旗）時重建
     - GPU 加速: 使用 OpenGL 硬體加速

   儲存格式

   JSON 格式儲存座標陣列 + 元資料，可選擇性儲存到 data/track_layouts/

   這是一個非常高效的設計，結合了物理真實性和效能最佳化！
